# 💸 Calculadora de Salário

> Cálculo simples e rápido do salário líquido com base em alíquotas.

---

## ✨ Visão Geral

Script em **JavaScript** que recebe o salário bruto e retorna:

* 💰 Salário bruto
* 📊 Alíquota aplicada
* 🧾 Imposto
* 🪙 Salário líquido

---

## ⚖️ Regras

```
Até R$ 5.000,00      → Isento
R$ 5.000,01 – 6.500  → 7,5%
R$ 6.500,01 – 8.000  → 15%
R$ 8.000,01 – 10.000 → 22,5%
Acima de R$ 10.000   → 27,5%
```

---

## 🚀 Execução

1. Rode o código no navegador
2. Digite o salário
3. Veja o resultado no alerta

---

## 🧠 Lógica

```
salarioLiquido = salarioBruto - imposto
```

---

## 📌 Saída

```
Relatório de salário:
Bruto: R$ 8000.00
Alíquota: 22,5%
Imposto: R$ 1800.00
Líquido: R$ 6200.00
```
